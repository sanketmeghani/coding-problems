# Rotated Array

Find second smallest integer in given array

Example input 1:

`[0]`

Example output 1:

`0`

Example input 2:

`[0, 1]`

Example output 2:

`1`

Example input 3:

`[0, 2, 1]`

Example output 3:

`1`
